¿Qué son los sitios web?

Un sitio web es un conjunto de archivos electrónicos y páginas web referentes a un tema en particular, incluyendo una página inicial de bienvenida generalmente denominada home page, a los cuales se puede acceder a través de un nombre de dominio y dirección en Internet específicos. El World Wide Web, o simplemente Web como se le llama comunmente, está integrado por sitios web y éstos a su vez por páginas web. La gente suele confundir estos términos, pero un sitio web es en realidad un conjunto de páginas web.

Tipos de sitios

Los juegos con los que va encontrándose la persona a medida que se desarrolla corresponden a distintas clases:

  • Web Estática. La información no puede variar, con lo cual se constituye en un recurso sencillo y rápido de aplicar para obtener presencia en el mundo de Internet. Muchas compañías digitales que comienzan a incorporarse a la era de las plataformas webs deciden utilizar este desarrollo en sus páginas ya que no actualizarán contenido frecuentemente.

  • Web dinámico. La actualización de este estilo de web se realiza de manera veloz y simple ya que se alimenta de aquellos datos que se almacenan en nuestras bases de datos. Si bien su desarrollo es más complejo, son las indicadas cuando debemos cambiar la información asiduamente. Si debes hacer un rediseño web, no dudes en mejorarlo con una ¡web dinámica!.
